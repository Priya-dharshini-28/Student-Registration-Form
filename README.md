This project is a simple HTML based Student Registration Form.
It allows users to enter and submit basic student details such as
-Name
-Age
-Email
-Department
-Gender
After submitting the form,the user is redirected to a Success page.
TECHNOLOGIES USED:
-HTML(form design)
-CSS(basic styling-optional)
FEATURES:
-input fields with placeholders
-Radio button selection for gender
-Reset button to clear the form
-Submit button to navigate to the Success page.
